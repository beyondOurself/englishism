# Unit 76 some 与 any 比较

## some 与 any 比较

## some

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/6109ac139753ddfa.jpg)


some 用于肯定句中：

□ I'm going to buy some clothes.

□ There's some ice in the fridge.

□ We made some mistakes.

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/e4ad61f526b63c83.jpg)


any 用于否定句中：

□ I'm not going to buy any clothes.

□ There isn't any milk in the fridge.

□ We didn't make any mistakes.

## any 与 some 在疑问句中的用法

大多数疑问句中（但不是所有疑问句中）用 any:

□ Is there any ice in the fridge?

□ Has he got any friends?

□ Do you need any help?

通常用 some 而不用 any 的情况：

主动提供某物 (Would you like ... ?)

□ A: Would you like some coffee?

B: Yes, please.

或要求某物 (Can I have ... ? 等)

□ A: Can I have some soup, please?

B: Yes. Help yourself.

□ A: Can you lend me some money?

B: Sure. How much do you need?

Have you got any money?

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/b48b8ebb46cbab8a.jpg)


Would you like some coffee?

## some 与 any 有时不带名词

□ I didn't take any photographs, but Jane took some. (= some photographs)

□ You can have some coffee, but I don't want any. (= any coffee)

□ I've just made some coffee. Would you like some? (= some coffee)

□ 'Where's your luggage?' 'I haven't got any.' (= any luggage)

□ 'Are there any biscuits?' 'Yes, there are some in the kitchen.' (= some biscuits)

## something / somebody (或 someone)

□ She said something.

anything / anybody (或 anyone)

□ I saw somebody (或 someone).

□ She didn't say anything.

□ Would you like something to eat?

□ Somebody's at the door.

□ I didn't see anybody (或 anyone).

□ Are you doing anything tonight?

□ Where's Sue? Has anybody seen her?
