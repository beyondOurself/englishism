## if 用于句首或句中：

## if 用于句首

If we go by bus, it will be cheaper.

If you don't hurry, you'll miss the train.

If you're hungry, have something to eat.

If the phone rings, can you answer it, please?

## if 用于句中

## 在对话中，常将 if 部分单独使用：

□ 'Are you going to the concert?' 'Yes, if I can get a ticket.'

## If you see Ann tomorrow ... 等

if 之后常用一般现在时（而不用 will），可说 if you see ...（而不用 if you will see）：

□ If you see Ann tomorrow, can you ask her to call me?

□ If I'm late this evening, don't wait for me. (而不用 If I will be)

□ What shall we do if it rains? (而不用 if it will rain)

□ If I don't feel well tomorrow, I'll stay at home.

## if 与 when 比较

if I go out 表示 “如果我要出去”（不确定）:

□ A: Are you going out later?

□ B: Maybe. If I go out, I'll close the windows.

## when I go out 表示 “当我出去时”（肯定）：

□ A: Are you going out later?

□ B: Yes, I am. When I go out, I'll close the windows.

## 请比较 when 与 if:

□ When I get home this evening, I'm going to have a shower.

□ If I'm late this evening, don't wait for me. (而不用 When I'm late)

□ We're going to play tennis if it doesn't rain. (而不用 when it doesn't rain)

## 可以说：

□ If he had the money, he would buy a car.

或 He would buy a car if he had the money.

I'd / she'd / they'd 等 = I would / she would / they would 等:

□ I don't know the answer. If I knew the answer, I'd tell you.

□ It's raining, so we're not going out. We'd get wet if we went out.

□ Jane lives in a city. She likes cities. She wouldn't be happy if she lived in the country

□ If you didn't have a job, what would you do? (但事实上你有工作)

□ I'm sorry I can't help you. I'd help you if I could. (但事实上我不行)

□ If we had a car, we could travel more. (但事实上我们没有车，所以不能经常旅游)

## if (I) was/were

可以说 if I/he/she/it was 或 if I/he/she/it were:

□ It's not a very nice place. I wouldn't go there if I were you. (或 ... if I was you)

□ It would be nice if the weather was better.
(或 ... if the weather were better)

I wouldn't go out if I were you.

□ What would Tom do if he were here?
(或 ... if he was here)
