# Unit 34 Would you like ...? I'd like ...

## Would you like ...? 表示“你想要……吗”。

## Would you like ...? 用来表示提供某物：

□ A: Would you like some coffee?

B: No, thank you.

□ A: Would you like a chocolate?

B: Yes, please.

□ A: What would you like, tea or coffee?
B: Tea, please.

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/a984a79daee59c23.jpg)


## Would you like to ...? 可用来表示邀请某人:

□ Would you like to go for a walk?

□ A: Would you like to have dinner with us on Sunday?

B: Yes, I'd love to. (= I would love to have dinner with you)

□ What would you like to do this evening?

I'd like ... 是 I want ... 的礼貌说法。I'd like = I would like:

□ I'm thirsty. I'd like a drink.

☐ (in a tourist office) I'd like some information about hotels, please.

□ I'm feeling tired. I'd like to stay at home this evening.

## Would you like ...? 与 Do you like ...? 比较

Would you like ... ? / I'd like ...
