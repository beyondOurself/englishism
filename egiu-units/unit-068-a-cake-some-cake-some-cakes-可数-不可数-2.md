## a/an 与 some 比较

a/an + 可数名词单数形式 (car/apple/shoe 等):

□ I need a new car.

□ Would you like an apple?

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/b08f757d30b9eea7.jpg)


an apple

some + 可数名词复数形式 (cars/apples/shoes 等):

□ I need some new shoes.

□ Would you like some apples?

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/a863ce84a8321d0d.jpg)


some apples

some + 不可数名词 (water/money/music 等):

□ I need some water.

□ Would you like some cheese?
(或 Would you like a piece of cheese?)

![image](https://raw.githubusercontent.com/beyondOurself/englishism/main/assets/egiu/f4dfe56ada11b6d9.jpg)


some cheese 或
a piece of cheese

请比较 a paper (= a newspaper) 与 some paper:

□ I want something to read. I'm going to buy a paper.

□ I want to make a shopping list. I need some paper / a piece of paper. (而不用 a paper)

## 应注意：

advice bread furniture hair information news weather work

这些名词通常不可数，因此不能说 a/an ... (a-bread, an advice)，也不能用复数形式 (advice, furnitures 等)。

□ Can I talk to you? I need some advice. (而不用 an advice)

□ I'm going to buy some bread. (而不用 a bread)

□ They've got some very nice furniture in their house. (而不用 furnitures)

□ Silvia has got very long hair. (而不用 hairs)

□ I'd like some information about hotels in London. (而不用 informations)

□ Listen! I've just had some good news. (而不用 a good news)

□ It's nice weather today. (而不用 a nice weather)

□ 'Do you like your job?' 'Yes, but it's hard work.' (而不用 a hard work)

可以说 a job (但不用 a work):

□ I've got a new job. (而不用 a new work)
