# Unit 32 should

## should + 动词原形

(should do / should watch 等):

<table><tr><td>I/we/you/they he/she/it</td><td>should shouldn't</td><td>do stop go watch 等</td></tr></table>

you should do something 表示 “做某事是好的或是对的”：

□ Tom doesn't study enough. He should study harder.

□ It's a good film. You should go and see it.

□ When you play tennis, you should always watch the ball.

you shouldn't do something 表示 “做某事是不好的”。

shouldn't = should not:

□ Tom shouldn't go to bed so late.

□ You watch TV all the time. You shouldn't watch TV so much.

## I think 常与 should 连用：

肯定句用 I think ... should ...:

☐ I think Lisa should buy some new clothes.

(= I think it is a good idea.)

□ It's late. I think I should go home now.

□ A: Shall I buy this coat?

B: Yes, I think you should.

否定句用 I don't think ... should ...:

□ I don't think you should work so hard.

(= I don't think it is a good idea.)

□ I don't think we should go yet. It's too early.

疑问句用 Do you think ... should ... ?:

□ Do you think I should buy this hat?

![image](https://gitee.com/shencanlong/englishism/raw/main/assets/egiu/460bf1485210fcac.jpg)


□ What time do you think we should go home?

must 比 should 语气更强烈：

□ It's a good film. You should go and see it.

□ It's a fantastic film. You must go and see it.

也可用 ought to 代替 should:

□ It's a good film. You ought to go and see it. (= you should go)

☐ I think Lisa ought to buy some new clothes. (= Lisa should buy)

I have to (do something) 表示 “我必须或有义务（做某事）”：

<table><tr><td>I/we/you/they have</td><td rowspan="2">to do to work to go to wear 等</td></tr><tr><td>he/she/it has</td></tr></table>

□ I'll be late for work tomorrow. I have to go to the dentist.

□ Jane starts work at 7 o'clock, so she has to get up at 6.

☐ You have to pass a test before you can get a driving licence.

表示过去 (yesterday / last week 等) 时，用 had to ...:

□ I was late for work yesterday. I had to go to the dentist.

□ We had to walk home last night. There were no buses.

## 在疑问句与否定句中用 do/does（现在时）与 did（过去时）：

现在时

<table><tr><td>do I/we/you/they</td><td rowspan="2">have to ... ?</td></tr><tr><td>does he/she/it</td></tr></table>

<table><tr><td>I/we/you/they don't</td><td rowspan="2">have to ...</td></tr><tr><td>he/she/it doesn't</td></tr></table>

过去时

<table><tr><td>did</td><td>I/we/you/they he/she/it</td><td>have to ... ?</td></tr></table>

<table><tr><td>I/we/you/they he/she/it</td><td>didn't have to ...</td></tr></table>

□ What time do you have to go to the dentist tomorrow?

□ Does Jane have to work on Sundays?

□ Why did they have to leave the party early?

I don't have to (do something) 表示 “我没有必要（做某事）”：

□ I'm not working tomorrow, so I don't have to get up early.

□ Ian doesn't have to work very hard. He's got an easy job.

□ We didn't have to wait very long for the bus – it came in a few minutes.
